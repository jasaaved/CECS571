1. Download all the files under the 'Project3' folder

2. Extract the apache-jena-3.14.0.zip to your hard drive along with the other files under the Project3 folder

3. Open the file 'apache-jena-fuseki-3.14.0' on command prompt

4. Run the command 'fuseki-server --update --mem /ds' to create a dataset named ds in memory

5. The server should now start running on your local host. The default port is usually 3030

6. Type on your local browser 'localhost:3030' and the page should display

7. Click on the 'add data' button to upload the datasets on the server

8. Click on select files button and upload the 'accidents.rdf', 'EmploymentRates.rdf' and 'Connecticut_Weather.owl' from the folder 'Project3'

9. Once the dataset is uploaded to the server, click on the query tab beside the upload files tab

10. Enter the queries in SPARQL format in the textbox provided 

11. The queries we came up with for the datasets are provided in the Queries document

12. Click the run button for the results to be displayed below
