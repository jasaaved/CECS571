This program converts a CSV dataset of NASA Labs/Facilities into RDF format.

To run: You only need to change the "inPath" and "outPath" to whichever
directory the "NASA_Labs_Facilities.csv" file is located in on your machine.
Needless to say, the Main.java class starts the program. Thanks.
